### Fibonacci Sequence
fibo(n) returns the fibonacci sequence of an integer, n.

The following are test cases of fibo(n):

fibo(0) == 0
fibo(1) == 1
fibo(2) == 1
fibo(3) == 2
fibo(4) == 3
fibo(5) == 5
fibo(10) == 55
